# Example data

Overview of example data used to test the app. For now not made publicly available.

## Standard batch growth experiment without dilution

- OD data: `batch_example/example_batch_data_od_readings.csv`
- OD calibration data: `data/batch_example/example_batch_data_od_readings_calibration.csv`

## Standard growth experiment with dilution (turbidostat mode)

- OD data: `turbidostat_example/example_2-Pio_Experiment_dilution_events.csv`
- Event (meta) data: `turbidostat_example/example_2-Pio_Experiment_dilution_events.csv`

## Second turbiostat experiment with varying dilution rates
